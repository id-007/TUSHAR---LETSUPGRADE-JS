Data-types in javaScript

  • String - represent sequence of characters
  • boolean - return a boolean value that will be either true of false
  • number - used to represent a numeric value
  • null - used to represent null value(no value)
  • undefined - used when we dont know the value
  • object - used as instance through which we can access members
  • array - used to group similar values
